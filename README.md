# Voting System — Java

A Java-based Voting System application built using Object Oriented Programming concepts. This project simulates a basic voting process where candidates can be registered, votes can be cast, and results can be displayed.

---

## Features

- Register candidates for the election
- Cast votes for a selected candidate
- Prevent duplicate voting
- Display total votes for each candidate
- Announce the winner based on highest votes

---

## Concepts Used

- Classes and Objects
- Constructors
- Encapsulation (private data members, getters/setters)
- Arrays and loops
- Conditional logic
- Input validation

## How to Run

1. Open **Eclipse IDE**
2. Import the project — `File → Import → Existing Projects into Workspace`
3. Select the project folder
4. Run the main file — `Right click → Run As → Java Application`
